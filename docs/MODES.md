# Game Modes

```text
╔════════════╦════════════════════╦════════════════════╗
║ Mode       ║ Map Shape          ║ Match Pressure     ║
╠════════════╬════════════════════╬════════════════════╣
║ 1v1 Duel   ║ compact arena      ║ fewer minions      ║
║ 3v3 Clash  ║ full arena         ║ full waves + teams ║
╚════════════╩════════════════════╩════════════════════╝
```

## 1v1 Duel

- Smaller arena bounds.
- Two minions per wave.
- Shorter wave interval.
- Smaller, faster Shrine objective.
- One enemy bot.
- Closer camera.

## 3v3 Core Clash

- Full arena bounds.
- Three minions per wave.
- Standard wave interval.
- Larger Shrine objective.
- Two allied bots and three enemy bots.
- Wider camera for teamfight readability.
